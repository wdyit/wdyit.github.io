个人想法

很早就想拥有个自己的博客，不需要也不可能比博客网站的内容好，但只是拥有自己的地盘，说明我在寻找自己的方向，博客慢慢记录着我成长的内容，从0到1的回忆，写博客，也是消化理解自己浑沦吞枣得到的知识，知识永远都学不完，但是现在至少证明我在努力得到，而不是索取。

PS：我是iOS开发出身，所以教程是mac 系统的，Windows系统的搭建方式相差无几

Hexo简介

Hexo是一个开源的静态博客生成器，底层用 Node.js 编写。
博客是一个个人网站最好的诠释，技术分享的圣地，只要简单干净利落就好，重要的是内容，是养分，而不是花哨。
静态博客编译后是html页面，很容易找到托管的网站，例如GitHub，osChina等。
hexo需要很熟悉git的命令才能部署，建议大家先了解git

准备

  1. 需要github账号，可去 Github 官网下载一个
  2. node.js 、npm 自行百度了解相关知识
  3. markdown 编辑器，个人推荐使用 mweb
  4. 域名： 如果想要自己的博客可用自己域名去访问，可以去申请个域名

GitHub操作

创建仓库
  1. 去 Github 主页创建一个仓库，仓库的名字要跟你创建的用户名一样，例如你的名字是wdyit, 仓库名字就是”wdyit.github.io”, 用 你的名字.github.io来命名, 以后可以直接访问 “http://你的名字.github.io"来访问你的博客
